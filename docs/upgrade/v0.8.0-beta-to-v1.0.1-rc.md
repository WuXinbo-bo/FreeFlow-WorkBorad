# FreeFlow 从 v0.8.0-beta 升级到 v1.0.1-rc 说明

## 1. 升级目标

- 让旧版 `v0.8.0-beta` 用户在安装 `v1.0.1-rc` 后，尽量延续原有画布、设置、会话和模型配置
- 将教程画布、最近画布、首次说明与主交互链路切换到新版逻辑

## 2. 推荐升级方式

1. 关闭当前正在运行的旧版 FreeFlow
2. 保留用户目录 `%USERPROFILE%\FreeFlow`
3. 直接运行新版安装包 `FreeFlow-v1.0.1-rc-x64.exe`
4. 使用默认安装目录覆盖安装
5. 首次启动后检查最近画布、设置与模型配置是否正常延续

## 3. 默认保留的数据

- `%USERPROFILE%\FreeFlow\AppData`
- `%USERPROFILE%\FreeFlow\CanvasBoards`

通常会延续的数据包括：

- 画布文件
- UI 设置
- 模型提供商配置
- 模型画像
- 会话记录
- 权限配置

## 4. 升级后需要重点确认的内容

- 是否直接打开最近画布，而不是反复进入教程画布
- 首次说明与教程中心是否按新版逻辑正常工作
- 旧画布元素是否能正常加载
- 导出 PDF / PNG 是否正常
- AI 镜像工作区是否能正常使用

## 5. 已知说明

- 教程本地状态版本已提升为 `1.0.1-rc`
- 升级后教程进度可能按新版教程逻辑重新计算
- 这不影响画布文件、设置和会话等主要业务数据
