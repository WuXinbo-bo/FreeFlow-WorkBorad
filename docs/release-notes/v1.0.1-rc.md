# FreeFlow v1.0.1-rc Release Notes

## Release Positioning

`v1.0.1-rc` is the current release-candidate packaging target for FreeFlow. This version focuses on release wrap-up, metadata unification, onboarding refinement, export workflow stabilization, and final desktop regression before shipping a new package.

## Highlights

- Desktop packaging metadata aligned to `v1.0.1-rc`
- Startup tutorial intro flow added and stabilized for first launch
- Tutorial center onboarding entry refined for new users
- Canvas export chain expanded and stabilized across PDF / PNG / capture flows
- Canvas background pattern options completed
- Rich text sizing and export boundary behavior further tightened

## Upgrade Notes

- Existing `v0.8.0-beta` users should keep their user data under `%USERPROFILE%\\FreeFlow`
- Existing `v1.0.0-rc` users can upgrade in place with the new installer
- Current compatibility base is designed to preserve:
  - UI settings
  - model provider settings
  - session history
  - permissions
  - clipboard queue
  - existing board file structure

## Known Release Risks Before Final Packaging

- First-launch onboarding and tutorial entry still need full desktop regression on packaged builds
- Export flows should be rechecked on oversized boards in real desktop packaging
- Final release artifact naming and visible version strings must remain fully aligned with `v1.0.1-rc`
