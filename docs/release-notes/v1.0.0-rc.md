# FreeFlow v1.0.0-rc Release Notes

## Release Positioning

`v1.0.0-rc` is the release-candidate milestone for FreeFlow. This version focuses on stabilizing the desktop shell, finishing the main canvas interaction chain, reorganizing the tutorial system, and preparing the project for formal public release.

## Highlights

- Desktop shell free-layout architecture stabilized
- Global overlay and window shape pipeline consolidated
- Recent board and tutorial board startup logic rebuilt
- Canvas global search added
- Rich text editing enhanced with large-font support
- Canvas structured copy / paste chain reworked
- Tutorial center reorganized with shortcut quick-reference entry
- AI mirror top controls and embedding workflow stabilized

## Upgrade Notes

- Existing `v0.8.0-beta` users should keep their user data under `%USERPROFILE%\\FreeFlow`
- Current compatibility base is designed to preserve:
  - UI settings
  - model provider settings
  - session history
  - permissions
  - clipboard queue
  - old board file structure
- Final release quality still depends on full pre-release regression

## Known Release Risks Before Final Packaging

- Canvas structured copy / paste and external clipboard switching still require focused regression
- Tutorial local state may not fully preserve progress across version boundary changes
- Packaging metadata and release artifact naming must remain fully aligned with `v1.0.0-rc`
