## Priority Rules

- Settings priority: persisted user settings from backend/AppData > Electron startup context > migrated legacy project settings > browser localStorage cache > code defaults.
- Recent canvas priority: `startup.initialBoardPath` > startup UI settings `canvasLastOpenedBoardPath` > remote `/api/ui-settings` value > browser cache > default board path. Prefer `.freeflow`; treat `.json` only as legacy import input.
- Layout priority: saved workbench preferences > startup context preferences > normalized defaults. Panel localStorage size/collapse keys must not override explicit habit defaults after save/apply.
- Theme priority: persisted UI settings theme fields > frontend normalized theme defaults.
